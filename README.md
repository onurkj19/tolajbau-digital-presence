# TOLAJBAU – Digital Presence

Website-u zyrtar i TOLAJBAU – Gerüstbau, Abbruch & Entsorgung | Schweiz.

## Teknologjitë

- React + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

## Instalimi

```sh
# Klono repo-n
git clone <YOUR_GIT_URL>

# Shko tek direktoria e projektit
cd tolajbau-digital-presence

# Instalo varësitë
npm install

# Starto serverin e zhvillimit
npm run dev
```

## Scripts

- `npm run dev` – Starto serverin e zhvillimit
- `npm run build` – Ndërto për prodhim
- `npm run preview` – Shiko ndërtimin e prodhimit
- `npm run lint` – Kontrollo kodin me ESLint
- `npm run test` – Ekzekuto testet
